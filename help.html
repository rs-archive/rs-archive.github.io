How To Help

If you ever played RuneScape on a computer, the full game files would be downloaded. So any old computer, laptop, hard drive, or backup that you might have used is worth searching for files.

We recommend using our open source "RuneScape Cache Finder" tool that can be downloaded here. This will automatically scan the drive for all the relevant files, and combine them into a file you can upload (no personal data whatsoever will be included).

You can then upload it to a filesharing website such as Easyupload, and either share the link in our Discord server, or submit it anonymously to this Google form.

For more information about how the cache is stored, visit this page.
